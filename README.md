# 摩擦力・振幅・湿度データ解析アプリ

このアプリは、CSVファイルから摩擦力、振幅、湿度データを解析し、摩擦係数、相対振幅、相対湿度を求めるためのPythonアプリケーションです。GUIを用いてフォルダを選択し、解析結果をグラフで表示します。

## 必要なライブラリ

- pandas
- numpy
- matplotlib
- scipy
- PySimpleGUI
- glob
- re

## 使い方

1. 必要なライブラリをインストールします。
2. アプリケーションを起動します。
3. 解析したいフォルダを選択します。
4. プログラムが実行され、解析結果がグラフで表示されます。

## 注意事項

- CSVファイルはNR-500から出力されたものであることを確認してください。
